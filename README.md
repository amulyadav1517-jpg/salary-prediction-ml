# Salary Prediction using Machine Learning

This project uses Linear Regression to predict salary based on years of experience.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Workflow
1. Load dataset
2. Train Linear Regression model
3. Predict salary
4. Visualize results

## ▶️ How to Run

Install required libraries:

pip install pandas scikit-learn matplotlib

Run the Python file:

python salary_prediction.py
